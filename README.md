 A AI project for classifying American Sign Language (ASL) letters (A-Z) from images using CNNs & Transfer Learning.

Includes data preprocessing, model training (ResNet50, Efficient NetBO, InceptionV3), performance evaluation, and a GUI for real-time predictions.

Team project for the Al Skills course.
